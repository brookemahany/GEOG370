This map uses the Equal Count (Quantile) method of classifying data in QGIS. The Quantile method of classification divides up the data so that each class as an equal amount of features. The pros of this method are that it neatly divides the data, and it can be good for evenly distributed data. However, there is not a lot of nuance in the sense that it can group a wide range of values into a single class. It might also put similar values in different classes.

This map uses the Equal Interval method of classifying data in QGIS. The Equal Interval method puts the data into different classes with the same ranges. For example, in the Quantile map, there is a class from 3-5 (range of 2) and the next class is from 5-6.2 (range of 1.2). However, all the classes in this map have a range of 1.6. The pros of this method are that it easily allows you to easily show continuous data. The disadvantage of this classification system is that there also might be too many values in a class. This is good for when you have a lot of data which several large differences in values. However, this data may not be good for differentiating extrenes from each other, and may put a wide range of higher numbers, for example, in the same class.
